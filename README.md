## 🚀 Projeto em Destaque

### Banking Microservices

Projeto focado na simulação de um fluxo de transações financeiras em arquitetura distribuída, utilizando comunicação assíncrona e observabilidade como pilares centrais.

**Stack:** Java 21 • Spring Boot • Kafka • Redis • MongoDB • Docker • Prometheus • Grafana • Camunda BPMN

**Principais conceitos aplicados:**
- Construção de fluxo transacional em arquitetura orientada a eventos  
- Uso de mensageria para desacoplamento entre serviços  
- Orquestração de processos com Camunda BPMN  
- Observabilidade com análise métricas de negócio e performance  
- Processamento distribuído e consistência de dados
